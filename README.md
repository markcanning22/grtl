# Setup

1. Clone or download the code to your PhpStormProjects folder
    ```sh
    git clone git@github.com:markcanning22/grtl.git ./
    ```
2. Install dependencies
    ```sh
   cd server
   npm install
   cd client
   npm install
   ```
3. Start the server
    ```sh
    cd server
    npm run dev
    ```
4. Start the client
    ```sh
    cd client
    npm run dev
    ```
