# Setup

1. Clone or download the code to your PhpstormProjects folder
    ```sh
    cd ~/PhpstormProjects/
    git clone git@github.com:markcanning22/grtl.git grtl
    ```
2. Install dependencies
    ```sh
   cd grtl

   cd server
   npm install
   cd client
   npm install
   ```
3. Start the server
    ```sh
    cd server
    npm run dev
    ```
4. Start the client
    ```sh
    cd client
    npm run dev
    ```
